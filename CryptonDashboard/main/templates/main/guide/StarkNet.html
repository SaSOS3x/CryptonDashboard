StarkNet - Setup Guide

System Requirements

StarkNet Miner minimum:

CPU

4 cores (8 preferred)

RAM

4GB (8 preferred)

OS

Ubuntu 20.04 recommended, 18.04 should work

Storage

500GB

Network

50 Mbps upload/download

Note:

You will require more than these requirement in order to be competitive

Установка

Если у вас ещё нет alchemy аккаунта, выполните пункт 'Получение эндпоинтов', после этого перейдите к следующему шагу.

Замените YOUR_ALCHEMY_HTTP_ADDRESS на адрес полученный во втором пункте.

ALCHEMY=YOUR_ALCHEMY_HTTP_ADDRESS
echo 'export ALCHEMY='$ALCHEMY >> $HOME/.bash_profile

Для установки ноды выполните однострочный скрипт установки:

wget -O starknet.sh https://api.nodes.guru/starknet.sh && chmod +x starknet.sh && ./starknet.sh

Получение эндпоинтов

Для запуска starknet мы будем использовать ноды, которые предоставляет сервис Alchemy, для этого зарегистрируйтесь на сайте alchemy.com и создайте эндпоинты в своем личном кабинете.

Копируем адрес и переходим к пункту 'Установка'

Дополнительно

Посмотреть логи

journalctl -u empowerd -f -o cat

Рестарт ноды:

systemctl restart empowerd

Обновить ноду:

wget -O starknet_update.sh https://api.nodes.guru/starknet_update.sh && chmod +x starknet_update.sh &&

Удаление ноды:

systemctl stop starknetd
systemctl disable starknetd
rm -rf ~/pathfinder/
rm -rf /etc/systemd/system/starknetd.service
rm -rf /usr/local/bin/pathfinder
