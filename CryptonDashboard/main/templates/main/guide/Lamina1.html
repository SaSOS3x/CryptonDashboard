Lamina1 - Setup Guide

System Requirements

Lamina1 Miner minimum:

CPU

4 cores (8 preferred)

RAM

8GB (16 preferred)

OS

Ubuntu 22.04 recommended

Storage

200GB

Network

50 Mbps upload/download

Note:

You will require more than these requirement in order to be competitive

Установка

Воспользуйтесь этими командами для быстрой установки:

echo "deb [trusted=yes arch=amd64] https://snapshotter.lamina1.global/ubuntu jammy main" > /etc/apt/

Проверьте bootstraping, если он true, перейдите к следующему шагу:

lamina1.check-bootstrap.sh

Узнаём id своей ноды:

lamina1.get_my_nodeid.sh

Используйте веб-кошелек, чтобы застейкать свои токены на ID вашей ноды. На скриншоте ниже не достаточно токенов для добавления валидатора.

Дополнительно

Посмотреть логи:

sudo journalctl -u lamina1-node.testnet -f -o cat

Рестарт ноды:

systemctl restart lamina1-node.testnet

Используемые порты:

9650, 9671

Удалить ноду:

systemctl stop lamina1
systemctl disable lamina1
rm -rf ~/lamina1 ~/.lamina1 /etc/systemd/system/lamina1.service
