Aptos - Setup Guide

System Requirements

Aptos Miner minimum:

CPU

8 cores (32 preferred)

RAM

32GB (32 preferred)

OS

Ubuntu 20.04 recommended, 18.04 should work

Storage

300GB

Network

50 Mbps upload/download

Note:

You will require more than these requirement in order to be competitive

Установка

Установка длится более 10 минут, рекомендуется запускать в screen:

screen -S aptos

Воспользуйтесь нашим скриптом для быстрой установки:

wget -q -O aptos_dev.sh https://api.nodes.guru/aptos_dev.sh && chmod +x aptos_dev.sh && sudo /bin/bash aptos_dev.sh

Для мониторинга ноды можете воспользоваться нашим телеграм ботом.

wget -q -O aptos_dev.sh https://api.nodes.guru/aptos_dev.sh && chmod +x aptos_dev.sh && sudo /bin/bash aptos_dev.sh

Полезные команды

Посмотреть логи:

journalctl -u aptosd -f

Рестарт ноды:

systemctl restart aptosd

Запросить метрики:

curl 127.0.0.1:9101/metrics 2> /dev/null | grep aptos_state_sync_version

Обновление:

wget -q -O aptos_update.sh https://api.nodes.guru/aptos_update.sh && chmod +x aptos_update.sh && sudo /bin/bash aptos_up

Обновление:

ssystemctl stop aptosd
systemctl disable aptosd
rm -rf ~/aptos-core
rm -rf ~/.aptos
rm -rf /opt/aptos/
