Aleo - Setup Guide

System Requirements

Aleo Miner minimum:

CPU

16 cores (32 preferred)

RAM

16GB (32 preferred)

OS

Ubuntu 20.04 recommended, 18.04 should work

Storage

128GB

Network

50 Mbps upload/download

Note:

You will require more than these requirement in order to be competitive

Рекомендуется следовать официальной документации Aleo для запуска и поддержания работы ноды. Однако, если у вас возникли сложности, воспользуйтесь гайдом ниже.

Не рекомендуется устанавливать на виртуальные сервера WebTropia в связи со случаями блокировок, ставьте на Dedicated сервера без приставки Virtual.

Рекомендуется устанавливать и запускать snarkos под пользователем root.

Установка

Воспользуйтесь нашим скриптом для быстрой установки:

wget -q -O aleo_snarkos3.sh https://api.nodes.guru/aleo_snarkos3.sh && chmod +x aleo_snarkos3.sh && sudo /bin/bash ale

Полезные команды

Посмотреть данные вашего Aleo аккаунта.

!!! СОХРАНИТЕ ЭТИ ДАННЫЕ В БЕЗОПАСНОМ МЕСТЕ !!!

cat $HOME/aleo/account_new.txt

Посмотреть какой Private Key используется в вашем прувере.

grep "prover" /etc/systemd/system/aleo-prover.service | awk '{print $5}'

Проверить логи прувера:

journalctl -u aleo-prover -f -o cat

Проверить логи клиента:

journalctl -u aleo-client -f -o cat

Остановить aleo прувер и запустить aleo клиент.

systemctl stop aleo-prover
systemctl restart aleo-client

Запуск прувера(Уже запущен после установки).

systemctl stop aleo-client
systemctl restart aleo-prover
