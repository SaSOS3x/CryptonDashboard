Anoma - Setup Guide

System Requirements

Anoma Miner minimum:

CPU

4 cores (32 preferred)

RAM

16GB or 8GB (32 preferred)

OS

Ubuntu 20.04 recommended, 18.04 should work

Storage

120GB

Network

50 Mbps upload/download

Note:

You will require more than these requirement in order to be competitive

Установка

Воспользуйтесь нашим скриптом для быстрой установки:

wget -q -O anoma.sh https://api.nodes.guru/anoma.sh && chmod +x anoma.sh && sudo /bin/bash anoma.sh

Ждите завершения установки, после чего выполните команду:

source $HOME/.bash_profile

Проверить логи ноды можно следующей командой:

journalctl -u anomad -f

Создание ключа

Генерируем новый ключ командой:

anoma wallet key gen --alias YOUR_WALLET_NAME

После полной синхронизации отправляем транзакцию для инициализации учетной записи:

anomac init-account --alias YOUR_ACCOUNT_ALIAS --public-key YOUR_WALLET_NAME --source YOUR_WALLET_NAME

Запрос токенов

Запрашиваем токены с крана командой:

anoma client transfer --source faucet --target YOUR_ACCOUNT_ALIAS --signer YOUR_ACCOUNT_ALIAS --token XAN --amount 1000

Проверяем баланс:

anoma client balance --token XAN --owner YOUR_ACCOUNT_ALIAS

Создание валидатора

Эта команда сгенерирует ключи, необходимые для запуска валидатора (нужно будет ввести пароль):

anoma client init-validator --alias YOUR_VALIDATOR_NAME --source YOUR_ACCOUNT_ALIAS

После этого перезапускаем ноду:

sudo systemctl restart anomad

Теперь делегируем нашему валидатору еще токенов:

anoma client bond --source YOUR_ACCOUNT_ALIAS --validator YOUR_VALIDATOR_NAME --amount 950

Проверить все делегации можно командой:

anoma client bonds --owner YOUR_ACCOUNT_ALIAS

Полезные команды

Отобразить все известные ключи:

anoma wallet key list

Сгенерировать новый ключ:

anoma wallet key gen --alias YOUR_WALLET_NAME

Перечислить все известные номера:

anoma wallet address list

Создать новый адрес:

anoma wallet address gen --alias YOUR_NEW_WALLET_ALIAS
